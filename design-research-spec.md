## Research Objective

Research top 50 chat/AI apps and design-forward applications to identify patterns for Aurelius UI redesign. Focus on earthy, natural, textured aesthetic with elegant serif typography.

## Research Areas Per App

For each app, document:
1. **Header/Navigation** - Top bar design, back navigation, menu patterns
2. **Color Palette** - Primary colors, accents, dark/light mode approach
3. **Typography** - Font families, weights, hierarchy clarity
4. **Dynamic Elements** - Animations, transitions, loading states
5. **Spatial Layout** - Content density, breathing room, card usage
6. **Branding** - Logo treatment, brand colors, visual identity
7. **Chat-Specific** - Message bubbles, AI response styling, input fields

---

## Anti-AI-Derivative Checklist

Flag any designs that exhibit these AI-generic patterns:
- Inter font usage
- Purple/blue gradients
- Card with shadow + 12dp radius + white background
- Bottom nav with Home/Search/Notifications/Profile
- Skeleton loading with gray rounded rectangles
- "Clean, minimal, modern" aesthetic without substance
- Generic Material 3 implementation

## Screenshot Analysis Requirements

Use `mcp__MiniMax__understand_image` to analyze:
1. Home/chat list screens
2. Chat conversation screens
3. Settings/profile screens
4. Dark mode variants if available

**Analysis Prompt Template:**
```
Analyze the UI design of this app screenshot:
1. What is the overall design aesthetic? (warm/cool, light/dark, textured/flat)
2. What colors dominate? (list hex codes where visible)
3. What typography is used? (serif/sans-serif, weight distribution)
4. How is the header/navigation structured?
5. How are messages/messages displayed?
6. What makes this design distinctive vs generic?
7. What could inspire an earthy, natural aesthetic?
```

## Constraints

- **Earthy/Natural Aesthetic Target** - Avoid cool grays and sterile whites
- **Elegant Serif Typography** - Avoid Inter/Roboto defaults
- **Distinctive Over Generic** - Every pattern must pass anti-AI test
- **Texture/Grain Interest** - Look for subtle depth beyond flat design
